# 📱 Flutter Component & Constructor Learning

In this project, I learned how to create a component and use a constructor in Flutter.

I created a component called:

* MyButton

In this component, I added a constructor with:

* title
* subtitle

I used:

* required this.title
* required this.subtitle

so that I can pass values when using the component.

Now I can use this component anywhere in my app by just calling MyButton and passing title and subtitle.

For example:

* MyButton(title: 'logIn', subtitle: ' Button')
* MyButton(title: 'signUp', subtitle: ' Button')

I also learned that:

* I should use final for variables in widgets
* I should use required when using constructor parameters

In this project, I also used a constants file where I created my own colors and used them in the UI.

## 💡 Key Learning

Create component once → Use everywhere
